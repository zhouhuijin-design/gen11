# 根茎森林互动网站

一个融合视频叙事与实时交互的沉浸式艺术体验。

## 📁 文件结构

```
根茎森林/
├── index.html              # 主页面
├── README.md              # 说明文档
├── ml5.min.js             # ml5.js 人脸检测库（如需）
└── 视频文件/
    ├── 微信视频2026-04-07_194336_517.mp4   # 第一段：开场封面视频（必需）
    ├── video_b_nodes.mp4                   # 第二段：过渡视频（可选）
    └── 13420035759257714.mp4              # 第三段：AI守林人引导视频（必需）
```

## 🎬 视频流程

网站采用**三段式视频叙事**：

```
┌─────────────┐    视频A播放结束    ┌─────────────┐   视频B播放结束   ┌─────────────┐
│  欢迎界面   │ ──────────────────▶ │  过渡效果   │ ──────────────▶ │ AI守林人引导 │
│  点击进入  │                     │ (可选)      │                 │             │
└─────────────┘                     └─────────────┘                 └─────────────┘
                                                                        │
                                                             视频C播放结束
                                                                        ▼
                                                                 ┌─────────────┐
                                                                 │  回忆花园   │
                                                                 │ 🌿 互动体验 │
                                                                 └─────────────┘
```

### 第一段：开场封面视频
- 文件：`微信视频2026-04-07_194336_517.mp4`
- 时长：建议 5-15 秒
- 内容：森林主题开场动画

### 第二段：过渡视频（可选）
- 文件：`video_b_nodes.mp4`
- 时长：建议 3-8 秒
- 内容：抽象粒子/根茎流动效果
- 如果不提供此视频，网站会自动跳过

### 第三段：AI守林人引导
- 文件：`13420035759257714.mp4`
- 时长：建议 10-20 秒
- 内容：AI守林人引导游客进入回忆花园

## 🚀 使用方法

### 1. 准备视频文件

将视频文件复制到 `根茎森林` 目录：

```
d:\curor\根茎森林\
├── index.html
├── ml5.min.js (可选)
├── 微信视频2026-04-07_194336_517.mp4   ← 必需
└── 13420035759257714.mp4              ← 必需
```

### 2. 启动网站

**推荐使用本地服务器**（支持摄像头等功能）：

```bash
cd d:\curor\根茎森林

# Python 3
python -m http.server 8000

# 或 Node.js
npx http-server -p 8000
```

访问：`http://localhost:8000`

## 🔧 高级配置

### 开启 AI 聊天功能

在代码中找到并填入你的 DeepSeek API Key：

```javascript
const DS_API_KEY = 'sk-你的API密钥'; // 填入你的密钥
```

### 修改布局模式

在代码中修改 `PATTERNS` 数组：

```javascript
const PATTERNS = [
    { name: "永恒丛林", func: (i, n) => {...} },
    { name: "星空螺旋", func: (i, n) => {...} },
    { name: "记忆网格", func: (i, n) => {...} }
    // 添加更多布局...
];
```

### 修改主题颜色

```css
:root {
    --accent: #7cfc00;       /* 主色调 */
    --bg: #050a05;           /* 背景色 */

```

## 📝 注意事项

1. **视频文件**：确保必需的视频文件已放入同一目录
2. **网络连接**：p5.js 和 ml5.js 从 CDN 加载，需要网络连接
3. **摄像头权限**：首次访问时需要授权摄像头（回忆花园功能）
4. **HTTPS 要求**：摄像头功能在部分浏览器需要 HTTPS
5. **浏览器支持**：Chrome、Firefox、Safari、Edge 现代版本

## 🐛 故障排除

### 视频不播放
- 检查视频文件是否存在
- 确认视频格式（推荐 H.264 MP4）
- 浏览器可能阻止自动播放，需要用户点击

### 回忆花园不显示
- 检查 ml5.js 是否加载成功
- 查看浏览器控制台错误信息

### 摄像头无法访问
- 检查浏览器权限设置
- 使用 HTTPS 访问
- 确认没有其他程序占用摄像头
